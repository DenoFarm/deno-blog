A blog that you can set up just like Ghost (uses Node JS).

Powered by Deno JS.
<br>

### How to run using deno run
```
deno run --allow-net --allow-write --allow-read --allow-plugin --allow-env --unstable server.js
```

### How to run using denon
```
denon start
```
